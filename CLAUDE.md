# CLAUDE.md — HSA 프로젝트 작업 지침

이 문서는 Claude Code가 **HSA(Heroes Survival Arena)** 코드 작업 시 따라야 할 프로젝트 규약이다.
프로젝트 개요·개발 환경·게임 소개는 [README](README.md), 단계별 구현 순서는 [ROADMAP](Documentation/ROADMAP.md), 코딩 표준은 [CODING-STANDARD](Documentation/CODING-STANDARD.md) 문서를 참고한다.

> 사용자 응대 언어, git commit 승인 게이트 등 상시 개인 규칙은 글로벌 `~/.claude/CLAUDE.md`에 있다. 여기엔 중복 기재하지 않는다.

## 빌드 / 검증 정책 (중요)

- **Claude은 C++ 코드 작성·수정까지만 한다.** 빌드/컴파일/Live Coding/에디터 작업은 **사용자가 수행**한다.
- 코드 변경 후 "컴파일이 통과한다"고 단정하지 않는다. 빌드는 사용자 몫이며, 사용자가 전달하는 빌드 결과(에러)를 받아 다음 작업을 이어간다.

## 디버깅 / 프로파일링 정책 (중요)

- **디버깅이 필요하다고 판단되면 로그(`UE_LOG`·`print` 등)를 찍는 방식으로 추적하지 않는다.** 대신 **어떻게 디버깅할지 방안을 사용자에게 제시**한다 — 사용자가 디버깅·프로파일링 도구를 직접 운용한다.
- 제시할 방안은 구체적이어야 한다: 어떤 도구를 쓸지, 어디에 breakpoint를 걸지, 어떤 변수·call stack·메모리를 관찰할지, 어떤 조건(conditional breakpoint·watch)을 둘지, 무엇을 보면 가설이 검증/반증되는지.
- 도구 예: Rider debugger(breakpoint·watch·call stack·memory view), Rider 내장 profiler(dotTrace/dotMemory 연동), Unreal Insights, `stat` 명령군, GPU/CPU profiler 등. 상황에 맞는 도구를 골라 권한다.
- 진단을 위한 임시 로그 코드를 코드베이스에 추가하지 않는다. 부득이하게 코드 계측이 필요하면 먼저 사용자에게 이유와 함께 제안하고 승인을 받는다.

## 엔진 소스 참조

- UE API의 시그니처·동작·관용구가 불확실할 때는 추측하지 말고 **로컬 엔진 소스를 직접 분석**해 확인한다.
- 엔진 루트: `C:\Program Files\Epic Games\UE_5.7\Engine` — 엔진 코드는 `Engine/Source/`, 플러그인은 `Engine/Plugins/`에 있다.
- 이 경로는 **읽기 전용 참조 대상**이다 — 절대 편집하지 않는다.

## 단일 출처 원칙

같은 사실(상태·값·정의)을 **두 곳 이상에 중복 기재하지 않는다.** 중복은 한쪽만 갱신될 때 반드시 어긋나며, 그 불일치는 사후에야 발견된다. 한 곳을 **단일 원본** 으로 정하고, 다른 곳은 그 원본을 **참조/링크**한다.

- **문서**:
  - 진행 상태·완료 여부는 **한 곳에서만** 추적한다(예: 체크리스트). 다른 섹션은 상태를 다시 적지 말고 "상태는 §X 참조"로 가리킨다.
  - 코드/설정에서 **파생되는 사실**(의존성 목록, 활성 플러그인, 파일 경로 등)은 문서에 값으로 복사하지 말고 **원본 파일을 링크**한다. 꼭 요약해야 하면 "원본은 이 파일, 아래는 요약"임을 명시한다.
  - 같은 개념의 설명은 한 문서에 두고, 나머지는 링크한다(README ↔ ROADMAP ↔ CLAUDE.md 역할 분담).
- **코드**:
  - 상수·매직넘버·설정값은 **한 곳에 정의**(상수/`UPROPERTY`/`DataAsset`/config)하고 참조한다. 같은 값을 여러 곳에 하드코딩하지 않는다.
  - 튜닝 데이터는 `DataAsset`/`DataTable`을 원본으로 두고, 로직은 거기서 읽는다.
  - 이미 존재하는 함수·유틸·타입을 재사용한다. 같은 로직을 복붙해 두 번째 정의를 만들지 않는다.
- **갱신 규칙**: 원본을 바꾸면, 그 원본을 값으로 복사해 둔 곳이 있는지 점검한다. 복사본을 발견하면 고치지 말고 **참조로 바꿔 중복 자체를 제거**한다.
