# HSA

프로젝트 이름은 Heroes Survival Arena의 이니셜인 HSA 입니다. Unreal Engine 5 Gameplay Systems 프로그래밍 실습 목적 프로젝트 입니다.

## 게임 소개

- **이름**: 미정
- **장르:** third-person, 협동(co-op) 멀티플레이, PvE 액션 서바이벌.
- **특징** 
  - 각 player는 **영웅(hero)** 을 선택하며, 영웅마다 고유한 전투 방식을 가집니다.
  - 팀은 **아레나(arena)** 에 들어가 점점 강해지는 적 wave를 상대로 오래 버티는 것이 목표입니다.

## 핵심 게임 루프(Core Gameplay Loop)

```
┌─────────────┐      ┌──────────────┐      ┌────────────┐      ┌─────────────┐
│  웨이브 시작  │ ──▶ │  모든 적 처치  │ ──▶ │ 웨이브 종료 │ ──▶ │   휴식 시간   │
└─────────────┘      └──────────────┘      └────────────┘      └──────┬──────┘
       ▲                                                              │
       │                   다음 웨이브 — 난이도 상승                     │
       └──────────────────────────────────────────────────────────────┘
```

1. **웨이브 시작** — 적이 arena에 spawn됩니다.
2. **웨이브 클리어** — 팀이 모든 적을 처치합니다.
3. **웨이브 종료** — 짧은 **휴식 시간(rest phase)** 이 시작됩니다.
4. **휴식 시간** — player는 **상점(shop)** 에서 item이나 equipment를 구매하고 전열을 가다듬습니다.
5. **다음 웨이브** — **난이도가 올라간** 새 wave가 시작되고, 루프가 반복됩니다.

## 계획 중인 기능(Planned Features)

> 아래는 디자인 목표입니다. 프로젝트가 초기 단계라 대부분 **아직 구현되지 않았습니다.**
> 단계별 구현 순서는 [개발 로드맵(ROADMAP)](Documentation/ROADMAP.md)을 참고하세요.

- **영웅 선택(hero selection)** — 서로 다른 전투 방식을 가진 여러 영웅.
- **협동 멀티플레이(co-op multiplayer)** — replication 기반 networked play.
- **웨이브 시스템(wave system)** — 난이도가 점점 올라가는 적 wave.
- **휴식 시간 상점(shop)** — rest phase 동안 item과 equipment 구매.
- **아레나 서바이벌** — 최대한 많은 wave를 버티기.

## 기술 스택(Tech Stack)

- **Unreal Engine 5.7**
- **C++** (단일 runtime module: `HSA`)
- **타겟 플랫폼:** PC / Windows
